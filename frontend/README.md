# mesto-react
------
**Описание работы**
------
Место - интерактивная соцсеть с возможностью добавлять свои фотографии, изменять аватар, имя и описание профиля. 
В дальнейшем появится авторизация.
------
## Используемые технологии: 
1. ReactJS
2. HTML, CSS, JS
3. Flexbox, Grid Layout
4. Nested БЭМ
5. WebPack
6. API
